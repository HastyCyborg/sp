Views:

- Admin

  - Create job listings
  - Create custom fields for listings
  - View database
  - Export to CSV or Google Sheets. Customise csv outputs.

- Student
  - Fill forms and Apply to listings
  - Edit profile data
  - View past applications

Student Profile:

- MIS
- First Name
- Last Name
- Middle Name
- Contact Number. (Primary and Alternate)
- Branch
- Institute mail
- personal mail
- permanent address
- Current address
- Resume
- Current CGPA
- Active Backlog
- Aggregate %, HSC, SSC
